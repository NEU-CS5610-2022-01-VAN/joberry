<!-- FEATURES -->
## Features

- Backend Features
  - Users Table & API
  - Posts Table & API
  - Activities Table & API
  - Berries Table & API
  - Comments Table & API

- Frontend Features
  - Home Page
  - Post Detail Page
  - User Detail Page
  - New/Edit Post Page
  - Search Result Page
  - Sign In Page
  - Profile Page 
  - Profile Settings Page

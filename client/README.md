# Joberry Frontend

### File Structure
```
├── config
├── public
├── scripts
└── src
    ├── components
    ├── pages
    ├── stores
    ├── styles
    └── utils
```

### Project Basics
Running for development
- `npm install / yarn`
- configure .env files
- `npm/yarn start`

### Frontend Routes
```
/home
/discover
/profile
/profile/settings
/posts/:id
/posts/edit
```
